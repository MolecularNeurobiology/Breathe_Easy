Installation¶

Packaged Program Install¶

  1. For our most recently released version, visit our GitHub release page.

  2. Select our most recent release.

  3. In the asset section, download the packaged zip file.

  4. Once the file is downloaded, unzip the file.

Source Install¶

Install Python3¶

Install python here.

Install Dependencies¶

  1. Activate Python virtual environment to help manage package installation.

#Posix
python3 -m venv <venv>
source <venv>/bin/activate

#Windows
python3 -m venv <venv>
<venv>/Scripts/activate.bat

  1. Install dependencies.

pip install -r requirements.txt

  1. Dependencies for R should be installed and libraries initiated upon the first run of STAGG.

Launching Breathe Easy¶

From packaged version¶

  1. Inside the unzipped file, you’ll find a folder named BASSPRO-STAGG_QUIPPL.

  2. Double click the launcher.bat file in this folder to launch the GUI.

    1. The launcher.bat file can be moved to your Desktop or another easily accessible location on your computer to make the program easy to access later.

From source¶

#Posix
source venv/bin/activate
python3 scripts/GUI/MainGUImain.py

#Windows
<venv>/Scripts/activate.bat
python3 scripts/GUI/MainGUImain.py
