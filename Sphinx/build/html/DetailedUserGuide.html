The number of files and choices you need to make to run Breathe Easy may feel overwhelming at first. However, if you follow the step-by-step instructions provided below, you will slowly become comfortable and familiar with the process of running your data. While learning to use the software is involved, it is because we aim to offer the utmost flexibility so that Breathe Easy can be applied to any dataset you can throw at it.
